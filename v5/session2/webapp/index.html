Multilingual BPE
Tokenizer Experiments

Designing a 10,000-token BPE vocabulary across English · Hindi · Telugu · Kannada Wikipedia pages of India. Goal: minimize the fertility ratio spread to maximize the assignment score.

Scoring Formula
Score = 1000 Xmax − Xmin where Xi = total tokens / faithful units for language i
Faithful unit = one contiguous Unicode letter/mark/number run OR one visible non-space punctuation character
Data

Corpus Overview

India's Wikipedia page scraped in four languages via the Wikipedia REST HTML API and converted to faithful Markdown (preserving links, URLs, tables, references, and image captions).
